태그별 배치속성

태그별로 block속성과 inline속성으로 구분할 수 있음. block : 한 라인 전체를 차지함 -> 개행이 됨. width heigth 조정가능 inline : 라인에서 여러 태그를 표현가능 -> 개행되지않음 width height조정 불가능 크기는 라인에서 지정하는대로 처리함. 기본 태그의 배치속성을 변경할 수 있음 -> display속성을 이용
