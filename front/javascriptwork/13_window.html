윈도우객체

윈도우객체는 자바스크립트의 최상위객체로 모든 객체와 함수를 가지고 있음. browser가 열리면 자동으로 생성이 됨. 한개 브라우저 창 한개 window객체가 생성. 윈도우객체의 주요자식객체 BOM
location : 브라우저에서 URL창에 대한 정보를 가지고 있는 객체 navigator : 브라우저에 대한 정보를 가지고 있는 객체 screen : 사용자 화면에 대한 정보를 가지고 있는 객체 history : 페이지 이동 이력정보를 가지고 있는 객체 * 함수로 수정불가 DOM
document : html태그를 객체화한 객체
