jQuery적용하기

jQuery를 적용하는 방법에는 두가지가 있음
1. CDN방식 : http통신으로 파일을 제공하는 서버에서 받아오는것
2. 파일다운로드 : 정해진 경로에 js파일 다운해서 불러오는것

jquery활용방식

jquery를 이용할려면 주로 onload와 비슷한 속성인 ready()함수안에 넣어서 이용함.

jquery사용문법(DOM)

$("선택자").jquery함수명(매개변수); jQuery("선택자").jquery함수명(매개변수); jQuery는 chaining $("선택자").jquery함수명(매개변수).jquery함수함수명().jquery함수명().....;

Element를 가져오는 방법
$("선택자")||jQuery("선택자") => jquery방식의 element를 불러옴.

:first, :last, :even, :odd

선택자:first -> 선택자로 선택된 element중 0번 인덱스 태그
선택자:last -> 선택자로 선택된 elemeent중 마지막 인덱스 태그
선택자:even -> 선택자로 선택된 element중 짝수 인덱스 태그
선택자:odd -> 선택자로 선택된 element중 홀수 인덱스 태그

:eq :gt :lt

  • 선택자:eq(n) -> 선택된 태그중 n번 인덱스 태그
  • 선택자:gt(n) -> 선택된 태그중 n번 인덱스 보다 큰 인덱스를 가진 태그
  • 선택자:lt(n) -> 선택된 태그중 n번 인덱스 보다 작은 인덱스를 가진 태그
